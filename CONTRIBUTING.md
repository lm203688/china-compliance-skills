# Contributing to China Compliance Skills

Thank you for your interest in contributing! 🎉

## How to Contribute

### Add a New Skill

1. Fork this repository
2. Create a new directory under `skills/` with a descriptive name (e.g., `cn-privacy-policy-generator`)
3. Create a `SKILL.md` file following this structure:

```markdown
---
name: your-skill-name
description: "Clear description of what the skill does and when to use it. Include both English and Chinese trigger words."
---

# Skill Title

[Core methodology section]

## Workflow

[Step-by-step detection/analysis workflow]

## Output Format

[Structured output template]

## Examples

[2-3 practical examples with input/output]

## Related Skills

[Links to other skills in this repo]
```

4. Submit a Pull Request

### Improve an Existing Skill

1. Fork this repository
2. Make your changes to the existing SKILL.md
3. Submit a Pull Request with a clear description of what changed and why

### Report Issues

- Open a GitHub Issue with:
  - Which skill has the problem
  - What you expected vs. what happened
  - Example input that triggers the issue

## Quality Standards

Every skill should include:

- ✅ **Methodology section** — Not just "do this", explain WHY
- ✅ **Structured workflow** — Step-by-step process, not vague instructions
- ✅ **Output format** — Clear template for consistent results
- ✅ **Practical examples** — At least 2 with real input/output
- ✅ **Legal citations** — Reference specific laws and articles
- ✅ **Related skills** — Link to other skills in this repo
- ✅ **Safety notes** — Disclaimer that this is not legal advice

## Legal Accuracy

- Always cite the specific law and article number (e.g., 广告法第九条第(三)项)
- Include the penalty range for violations
- Note when rules have changed recently
- Mark uncertain areas clearly

## Language

- Skill content should be in **both Chinese and English** where practical
- Legal terms must use the official Chinese term with English translation
- Examples should use realistic Chinese content

Thank you for helping make Chinese content compliance more accessible! 🛡️
